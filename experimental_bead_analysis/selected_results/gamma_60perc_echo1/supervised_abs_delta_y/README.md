These selected figures come from the notebook-end supervised mobility experiment connected to the bead echo analysis.

In this experiment, DCIts was trained with bead-motion inputs such as `raw_xy`, `rel_xy`, and `dxy`, but the supervised target was the movement magnitude

```text
|delta raw_xy| = |raw_xy(t) - raw_xy(t-1)|.
```
