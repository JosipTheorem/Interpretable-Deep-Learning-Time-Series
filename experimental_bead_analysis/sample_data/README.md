# Sample Bead Data

This folder contains two tiny raw-data samples copied from the experimental Eva bead-tracking data so the bead-analysis code has a minimal runnable input.

Included samples:

- `lowrange echo 10 rad_s 1perc/individual_beads_echo1`: cluster 0 from the 1% amplitude case, 10 full-frame bead tracks.
- `highrange echo 10 rad_s 60perc/individual_beads_echo1`: cluster 0 from the 60% amplitude case, 10 full-frame bead tracks.

The complete raw dataset is not included in git. Ask before publishing these `.mat` files if redistribution permission is uncertain.
