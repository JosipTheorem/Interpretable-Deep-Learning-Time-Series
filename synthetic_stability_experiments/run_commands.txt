# Synthetic Stability Experiment Commands

Run these commands from the shared workspace folder after setup:

workspace/
|-- DCIts/
`-- Interpretable-Deep-Learning-Time-Series/

The experiment folder must be copied into:

DCIts/examples/synthetic_stability_experiments/

All default outputs are written under:

results/

Notes:
- Full runs use n_runs=10 and ts_length=20000 by default, so they can take a long time.
- Add --device cpu if CUDA is not available or if you want a CPU-only test.
- Use the quick test commands first to check that the setup works.
- Command-line arguments override values from the JSON config files.


===============================================================================
TASK 1: HIDDEN DRIVER / OBSERVED DRIVER
===============================================================================

Full default run:

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json

Default settings come from hidden_driver_config.json:
- A_values = 0.5, 1.0, 2.0
- P_values = 50, 200, 500
- rho_U_values = 0.0, 0.3, 0.7
- n_runs = 10
- output_dir = results/hidden_driver

Quick smoke test:

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --A-values 1.0 --P-values 200 --rho-values 0.3 --skip-control --n-runs 1 --ts-length 1000 --epochs 1 --output-dir "results/test_hidden_driver_single"

Main sweeps used for the thesis:

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --A-values 0.5,1.0,2.0 --P-values 200 --rho-values 0.3 --skip-control

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --A-values 1.0 --P-values 50,500 --rho-values 0.3 --skip-control

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --A-values 1.0 --P-values 200 --rho-values 0.0,0.7 --skip-control

Control cases:

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --only-control --rho-values 0.0,0.3,0.7

Control with A=0, rho=0, and sigma_U=0:

python DCIts\examples\synthetic_stability_experiments\hidden_driver_pipeline.py --config DCIts\examples\synthetic_stability_experiments\hidden_driver_config.json --output-dir "results/hidden_driver_no_U_control" --only-control --rho-values 0.0 --sigma-U 0.0

Main output layout:

results/hidden_driver/
|-- run_config.json
|-- summary_tables/
|   |-- all_case_summary.csv
|   |-- all_stable_links.csv
|   `-- all_lag_tables.csv
|-- main_sweep/
|   `-- A_..._P_..._rho_.../
|       |-- config.json
|       |-- generated_series.npz
|       |-- case_summary.csv
|       |-- time_series/
|       |-- comparison_heatmaps/
|       |-- observed_U/
|       |-- hidden_U/
|       `-- hidden_U_without_seasonal/
`-- control_A0/
    `-- A_0p0_rho_.../

Each observed/hidden case folder contains training_results.pkl, stable_links.csv,
lag_table.csv, false_positives.csv, and heatmap PDFs.


===============================================================================
TASK 2: REGIME CHANGE
===============================================================================

Full default run:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json

Default settings come from regime_change_config.json:
- B_values = [25,75], [50,150], [100,300], [50,2000]
- sigma_sets = low_snr, high_snr
- losses = mse
- n_runs = 10
- output_dir = results/regime_change

Quick smoke test:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --B-values 50:150 --sigma-set-names high_snr --n-runs 1 --ts-length 1000 --epochs 1 --output-dir "results/test_regime_change_single"

Run only high-SNR cases:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --sigma-set-names high_snr

Run only low-SNR cases:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --sigma-set-names low_snr

Run only PDF B values with high-SNR:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --B-values 25:75,50:150,100:300 --sigma-set-names high_snr

Run only long-regime case with high-SNR:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --B-values 50:2000 --sigma-set-names high_snr

Run loss comparison on high-SNR:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --sigma-set-names high_snr --losses mse,huber,smooth_l1,mae

Run longer training on high-SNR:

python DCIts\examples\synthetic_stability_experiments\regime_change_pipeline.py --config DCIts\examples\synthetic_stability_experiments\regime_change_config.json --sigma-set-names high_snr --epochs 300 --scheduler-patience 10 --early-stopping-modifier 3

|   |-- all_case_summary.csv
|   |-- all_regime_metrics.csv
|   |-- all_lag_sign_metrics.csv
|   |-- all_link_tables.csv
|   `-- all_classification_metrics.csv
`-- B_.../
    `-- sigma_.../
        |-- generated_series.npz
        |-- time_series/
        `-- loss_mse/
            |-- case_summary.csv

Each source_change / lag_change / sign_change folder contains:
