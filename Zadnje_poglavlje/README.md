# Additional DCIts Tasks

This folder contains the final synthetic experiments:

- `Zadatak1_pipeline.py`: hidden driver and observed driver experiments;
- `Zadatak2_pipeline.py`: regime-change experiments;
- `Zadatak3_pipeline.py`: smoothly time-varying coefficient experiments.

Each pipeline has a matching JSON configuration file and writes CSV tables plus PDF figures.

Use `komande.txt` for the exact command list used during development.
